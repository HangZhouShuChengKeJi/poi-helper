# poi-helper
[Apache poi](https://poi.apache.org/) 库辅助工具，提供更便捷的操作 word、excel 等的方法

# 使用
在 maven 项目中使用时，通过以下方式添加依赖：
```xml
    <dependency>
        <groupId>com.orange.opensource</groupId>
        <artifactId>poi-helper</artifactId>
        <version>${eventframework.version}</version>
    </dependency>
```
